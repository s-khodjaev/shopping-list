# Shopping List Project

## Description

This is a simple shopping list project that allows you to add items to your shopping list and mark them as completed.

## Features

- Add items to your shopping list
- Mark items as completed
- Delete items from your shopping list
- Clear your shopping list

## Requirements

- Node.js
- npm

## Installation

1. Clone the repository:

```bash
git clone https://github.com/s-khodjaev/shopping-list-project.git
```

2. Navigate to the project directory:

```bash
cd shopping-list-project
```

3. Install the project dependencies:

```bash
npm install
```

4. Start the project:

```bash
npm start
```

5. Open your browser and navigate to `http://localhost:3000`.

## Usage

1. Add items to your shopping list:

   - Click the "Add Item" button.
   - Enter the item name.
   - Click the "Add" button.

2. Mark items as completed:

   - Click the "Mark Completed" button.
   - Select the item to mark as completed.
   - Click the "Mark Completed" button.

3. Delete items from your shopping list:

   - Click the "Delete" button.
   - Select the item to delete.
   - Click the "Delete" button.

4. Clear your shopping list:

   - Click the "Clear" button.
